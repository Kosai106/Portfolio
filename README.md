## Building & running

To start up a local server, run the following commands.

```bash
npm install
npm run dev
```

Now go to [localhost:8080](http://localhost:8080/)

----

To build the project files, run the following commands.

```bash
npm install
npm run build
```

## Deploying

```bash
npm run deploy
```
